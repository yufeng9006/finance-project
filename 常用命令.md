# 常用命令

## 后端 (Laravel/PHP)

### 启动服务
```bash
cd backend
php artisan serve
```

### 数据库迁移
```bash
cd backend
php artisan migrate
```

### 数据库填充
```bash
cd backend
php artisan db:seed
```

### 创建模型/控制器/迁移文件
```bash
cd backend
php artisan make:model ModelName
php artisan make:controller ControllerName
php artisan make:migration create_table_name
```

## 前端 (Vue/Vite)

### 安装依赖
```bash
cd frontend
npm install
```

### 启动开发服务器
```bash
cd frontend
npm run dev
```

### 构建生产版本
```bash
cd frontend
npm run build
```

## 开发环境一键启动

项目提供了两个脚本用于快速启动开发环境：

1. 使用独立终端的方式（推荐）：
```bash
./start-services.sh
```

2. 使用单一命令同时启动前后端：
```bash
./start-dev.sh
```

按 `Ctrl+C` 可以停止服务。
