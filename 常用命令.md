# 常用命令清单

## 后端开发命令

### Laravel 命令
```bash
# 启动开发服务器
php artisan serve

# 创建控制器
php artisan make:controller NameController

# 创建模型
php artisan make:model Name

# 创建迁移文件
php artisan make:migration create_table_name_table

# 运行迁移
php artisan migrate

# 回滚迁移
php artisan migrate:rollback

# 重新运行所有迁移（删除并重新创建所有表）
php artisan migrate:fresh

# 填充数据
php artisan db:seed

# 创建填充器
php artisan make:seeder NameSeeder

# 清除配置缓存
php artisan config:clear

# 清除路由缓存
php artisan route:clear

# 清除视图缓存
php artisan view:clear

# 生成应用密钥
php artisan key:generate

# 运行队列
php artisan queue:work

# 创建资源控制器
php artisan make:controller API/ControllerName --model=ModelName
```

### 数据库相关命令
```bash
# 连接到 MySQL
mysql -u finance_user -p

# 连接为 root 用户
mysql -u root -p

# 导出数据库结构
mysqldump -u root -p --no-data home_finance_system > sql.sql

# 导出完整数据库
mysqldump -u root -p home_finance_system > home_finance_system_backup.sql
```

## 前端开发命令

### Vue 开发命令
```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## Xdebug 调试命令

```bash
# 检查 Xdebug 是否已安装
php -m | grep xdebug

# 查看 Xdebug 配置信息
php -i | grep xdebug
```

## Git 命令

```bash
# 提交所有更改
git add .
git commit -m "提交信息"

# 推送到远程仓库
git push

# 拉取最新代码
git pull

# 查看状态
git status

# 查看提交历史
git log
```
